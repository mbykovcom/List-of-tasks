List of Tasks - Методы API

  • http://localhost:5000/create_user - Создать нового пользователя [POST - запрос]

    • POST запрос с телом json

            {
            "login": "user1",
            "password": "pass",
        }

    Ответ

            {
            "status": 201,
            "message": "user user1 created"
        }
  • http://localhost:5000/tasks - Получение списка задач [GET - запрос]

    • Ответ

            {
            "tasks": [
                {
                    "deadline": "2020-03-12 14:00",
                    "description": "Description 1",
                    "done": false,
                    "id": 1,
                    "title": "Task 1"
                },
                {
                    "deadline": "2020-03-12 14:30",
                    "description": "Description 2",
                    "done": false,
                    "id": 2,
                    "title": "Task 2"
                }
            ]
        }
  • http://localhost:5000/create_task - Создать новую задачу [POST - запрос]

    • POST запрос с телом json

            {
            "title": "Task 3",
            "description": "Description 3",
            "deadline": "2020-03-12 15:00"
        }

    Ответ

            {
            "status": 201,
            "task": {
                "deadline": "2020-03-12 15:00",
                "description": "Description 3",
                "done": false,
                "id": 3,
                "title": "Task 3"
            }
        }
  • http://localhost:5000/tasks/id - Получение задачи по id [GET - запрос]

    • Ответ

            {
            "tasks": [
                {
                    "deadline": "2020-03-12 15:00",
                    "description": "Description 3",
                    "done": false,
                    "id": 3,
                    "title": "Task 3"
                }
        }
  • http://localhost:5000/done/id - Отметить задачу как выполненную [PUT - запрос]

    • Ответ

            {
            "message": "Task 3 was completed",
            "status": 200,
            "task": {
                "deadline": "2020-03-12 15:00",
                "description": "Description 3",
                "done": true,
                "id": 3,
                "title": "Task 3"
            }
        }
  • http://localhost:5000/delete_task/id - Удалить задачу [DELETE - запрос]

    • Ответ

            {
            "message": "Task 3 was deleted",
            "status": 200
        }
  • http://localhost:5000/delete_user - Удалить пользователя [DELETE - запрос]

    • Ответ

            {
            "message": "user user1 was deleted",
            "status": 200
        }
